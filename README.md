# 生活習慣管理システム

生活習慣の改善と日々のルーティンを効率的に管理するためのGitHubベースのシステムです。
GitHub ProjectsとIssues、GitHub Actions、そしてClineを活用して、習慣化をサポートします。

## 特徴

- 🎯 目標管理：長期的な習慣化目標（Epic）の設定と進捗管理
- 📝 デイリータスク：日々のルーティンを効率的に管理
- 📊 自動進捗レポート：毎日の達成状況を自動で記録
- 🔄 ワークフロー自動化：タスクの状態更新を自動化
- 📱 Cline連携：VSCode上で簡単にタスク管理

## セットアップ

詳細な手順は[セットアップガイド](docs/setup-guide.md)を参照してください。

1. リポジトリをクローン
2. GitHub Projectsの設定
3. ラベルとワークフローの設定
4. Clineでの操作開始

## 使用方法

1. Epicの作成
   - 新しい習慣化プログラムを開始する際に作成
   - 目標と期間を設定

2. デイリータスクの管理
   - 毎日のルーティンをイシューとして作成
   - 進捗を記録

3. 振り返りと改善
   - 自動生成される日報で進捗を確認
   - 定期的な振り返りで習慣を改善
